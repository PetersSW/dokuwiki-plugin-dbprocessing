====== TCS Inventar ======
===== Select =====
SQL-Abfrage für Tabelleninhalt //(Feldname bzw. Alias ist Spaltenüberschrift)//
<code sql select>
SELECT id AS "Inventarnr.", Beschreibung, Kategorie, Zugang, Abgang 
FROM `tcs22inventar` 
WHERE 1  
ORDER BY id;
</code>
